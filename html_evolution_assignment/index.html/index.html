History of HTML
HTML Evolution
  • Berners-Lee creates the first version of HTML in 1991.
  • HTML 1.0 then gets released to the public in 1993.
  • HTML 2.0 was created in 1995, allowing for more features such as heading, lists, images and more.
  • HTML 3.2 was created in 1997 with new features like creating tables and framesets
  • HTML 4.0 was developed and released in 1999. This new installment allowed coders to create containers, objects and buttons.
  • HTML in 1999 allowed for easier code to develop and adopted the Universal Character Set.
  • Later in the year 1999, HTML 4.01 was developed which allowed more support to multimedia options and scripting languages.
  • A rewrite of HTML known as XHTML was developed and became an offical standard in 2000.
  • HTML5 was in development in 2004 which took several years to fully finalize.
  • HTML5 was then finalized during the year 2014, becoming a fundemental landmark in the coding industry.
